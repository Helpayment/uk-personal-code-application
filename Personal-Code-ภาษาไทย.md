## Personal Code คืออะไร?

**Personal Code** คือหมายเลขประจำตัวเฉพาะที่ Companies House ออกให้หลังจากยืนยันตัวตนแล้ว เป็นข้อกำหนดทางกฎหมายตาม **พระราชบัญญัติอาชญากรรมทางเศรษฐกิจและความโปร่งใสของบริษัท 2023 (ECCTA 2023)**

ตั้งแต่ **18 พฤศจิกายน 2568** กรรมการและ PSC ทุกคนที่ยื่น Confirmation Statement (CS01) ต้องระบุ Personal Code ที่ผ่านการยืนยันแล้ว มิฉะนั้นจะถูกปฏิเสธ

## ใครต้องทำการยืนยัน?

- **กรรมการ** ของบริษัทอังกฤษ (Ltd, PLC)
- **บุคคลที่มีอำนาจควบคุมสำคัญ (PSC)** / ผู้ถือหุ้นเกิน 25%
- **เลขานุการบริษัท**, สมาชิก LLP

**ใช้บังคับกับชาวต่างชาติที่อาศัยอยู่นอกสหราชอาณาจักรด้วย**

## ทำไมเส้นทาง GOV.UK จึงไม่เหมาะสำหรับชาวไทย?

การยืนยันตัวเองผ่าน GOV.UK One Login ต้องการ:
- หนังสือเดินทางชีวมิติที่มีชิป NFC ที่ใช้งานได้
- ที่อยู่ในสหราชอาณาจักร
- การสแกนชิป NFC สำเร็จด้วยแอพมือถือ (มักล้มเหลวบ่อยครั้ง)

สำหรับกรรมการไทยส่วนใหญ่ที่ไม่มีที่อยู่ในสหราชอาณาจักร เส้นทางนี้**แทบจะเป็นไปไม่ได้**

**ในฐานะ ACSP เราสามารถ:**

✅ ไม่ต้องสแกนชิป NFC  
✅ ไม่ต้องมีที่อยู่ในสหราชอาณาจักร  
✅ ไม่ต้องเดินทางไปสหราชอาณาจักร  
✅ รับหนังสือเดินทางไทยและทุกสัญชาติ  
✅ ตรวจสอบเอกสารโดยเจ้าหน้าที่  
✅ 100% ออนไลน์จากประเทศไทย
