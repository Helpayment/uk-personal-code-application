## Personal Code とは？

**Personal Code** は、本人確認完了後に Companies House が発行する固有の識別番号です。**2023年経済犯罪・企業透明性法（ECCTA 2023）** により法的義務となっています。

**2025 年 11 月 18 日以降**、Confirmation Statement（CS01）を提出するすべての取締役・PSC は、確認済みの Personal Code を含める必要があります。含めない場合、申告は却下されます。

- **カンパニーセクレタリー**、LLP メンバー

**英国外在住の外国籍取締役・PSC も対象です。**

## なぜ GOV.UK ルートが日本人に向かないのか？

GOV.UK One Login による自己確認には以下が必要です：
- 機能する NFC チップ付き生体認証パスポート
- 英国の住所
- スマートフォンアプリによる NFC チップの読み取り（頻繁に失敗）

英国に住所を持たない日本人取締役にとって、このルートは**現実的でない**ことがほとんどです。

**ACSP（授権企業サービスプロバイダー）として、弊社は：**

✅ NFC チップスキャン不要  
✅ 英国の住所不要  
✅ 英国への渡航不要  
✅ 日本国パスポート対応  
✅ 人による書類審査  
✅ 日本からでも完全オンラインで申請可能

## 申請の流れ

1. **書類をオンラインで提出** — パスポートと住所証明書をアップロード
2. **ACSP チームによる審査** — 担当者が書類を確認
3. **Personal Code の登録** — Companies House に情報が記録されます
4. **Confirmation Statement を提出** — Personal Code を含めて CS01 を申告

---

## 締め切り

締め切りは個人によって異なります。次回の **Confirmation Statement（CS01）の提出期限**が実質的な締め切りです。

**2025 年 11 月 18 日以降に CS01 が期限を迎える場合**、それまでに確認済みの Personal Code が必要です。
