## 퍼스널 코드란?

**퍼스널 코드(Personal Code)** 는 신원 확인 완료 후 Companies House가 발급하는 고유 식별 번호입니다. **2023년 경제범죄 및 기업투명성법(ECCTA 2023)** 에 따라 법적 의무가 되었습니다.

**2025년 11월 18일부터** 확인 신고서(CS01)를 제출하는 모든 이사와 PSC는 확인된 퍼스널 코드를 포함해야 합니다. 포함하지 않으면 제출이 거부됩니다.

## GOV.UK 자가 신청이 한국인에게 어려운 이유

GOV.UK One Login을 통한 자가 확인을 위해서는 다음이 필요합니다:
- NFC 칩이 작동하는 생체인식 여권
- 영국 주소
- 스마트폰 앱으로 NFC 칩 스캔 성공 (자주 실패함)

영국에 주소가 없는 한국인 이사에게는 이 방법이 **현실적으로 불가능한 경우가 많습니다.**

**ACSP(승인된 기업 서비스 제공자)로서 저희는:**

✅ NFC 칩 스캔 불필요  
✅ 영국 주소 불필요  
✅ 영국 방문 불필요  
✅ 한국 여권 포함 모든 국적 수용  
✅ 담당자의 직접 서류 검토  
✅ 한국에서 100% 온라인으로 신청 가능
